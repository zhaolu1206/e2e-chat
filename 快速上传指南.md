## 🔑 最后一步：创建 Personal Access Token

GitHub 不再接受密码，需要使用 Token。请按以下步骤操作：

### 方法 1: 快速创建 Token（推荐）

1. **打开浏览器**，访问：https://github.com/settings/tokens/new
2. **填写信息**：
   - **Note**: `e2e-chat-upload`
   - **Expiration**: 选择 `90 days` 或更长
   - **勾选 `repo`**（完整仓库访问权限）
3. **点击** `Generate token`（绿色按钮）
4. **立即复制 Token**（格式：`ghp_xxxxxxxxxxxxx`）

### 方法 2: 手动创建

1. 登录 GitHub
2. 点击右上角头像 → `Settings`
3. 左侧菜单最下方 → `Developer settings`
4. `Personal access tokens` → `Tokens (classic)`
5. `Generate new token` → `Generate new token (classic)`
6. 填写信息并生成
